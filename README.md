# Bookapp

See the code live: https://fast-lake-81482.herokuapp.com

This is an app for adding and reviewing your favorite books.  It connects to the google books API.

Stack info:

Front-end:
- CSS grid
- Bootstrap
- EJS view

Back-end:
- MongoDB
- Express
- NodeJS
